uint16_t VoltMon_ReadVoltageProject_mV ( void  )

Read and condition the project supply voltage in millivolts.

Goal of the function

The purpose of this function is to provide a robust project-specific voltage measurement (in mV) by selecting/combining two internal voltage samples: an unfiltered value (supplyDcNotFiler_u16) and a filtered value (supplyDcFiler_u16).
