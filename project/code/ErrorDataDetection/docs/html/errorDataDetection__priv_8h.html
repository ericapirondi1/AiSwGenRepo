Detailed Description

Private declarations for the Error Data Detection module.

This header is intended for internal module usage only. It declares private constants, the CRC-8 lookup table, and internal static helper functions used by errorDataDetection.c.

Warning
This header shall not be included by external modules.

const uint8_t EDD_Crc8Table[256]

CRC-8 lookup table for polynomial 0x07.

This table is precomputed to allow CRC computation in O(N) time with minimal CPU load. It maps intermediate CRC values combined with the next input byte to the next CRC accumulator value.
